🎓 Student Management System

A full-stack web application developed using Python Flask, MySQL, SQLAlchemy, HTML, CSS, and JavaScript to manage students, teachers, attendance, exams, fees, and announcements in educational institutions.

🚀 Features

Authentication

- Teacher Registration & Login
- Admin Login
- Password Hashing
- Session Management
- Forgot Password Structure

Admin Features

- Dashboard Analytics
- Teacher Management
- Teacher Activity Tracking
- Online/Offline Status Monitoring
- Announcements Management
- View Teacher Profiles

Teacher Features

- Student Management
- Attendance Management
- Exam Management
- Fee Management
- Timetable Management
- View Announcements

Student Management

- Add Students
- Edit Students
- Delete Students
- Student Profile View
- Student Image Upload
- Student ID Card Generation

Attendance System

- Mark Attendance
- Attendance Reports
- Teacher-wise Attendance Tracking
- PDF Attendance Download

Examination System

- Add Exam Marks
- Pass/Fail Calculation
- Exam Reports

Fee Management

- Add Fee Records
- Track Paid/Pending Fees
- Fee Reports

Announcement System

- Admin can post announcements
- Teachers can view announcements on dashboard

⚙️ Installation

Clone Repository

git clone https://github.com/your-username/student-management-system.git
cd student-management-system

Create Virtual Environment

python -m venv venv

Activate Virtual Environment

Windows:

venv\Scripts\activate

Install Dependencies

pip install -r requirements.txt

Configure Database

Create MySQL Database:

CREATE DATABASE student_management;

Update database configuration in "config.py".

Run Application

python app.py

This system helps schools, colleges, and training institutes manage academic and administrative activities digitally, reducing paperwork and improving efficiency.
